# Button

## 簡介

`Button` 元件執行表單、對話方塊等中的操作，並支援多種尺寸、狀態等。

## 變體

Bootstrap 包括幾個按鈕變體，每個變體都有自己的語義目的，並添加了一些額外的功能以實現更多控制。

## Outline 按鈕

需要一個按鈕，但不需要它們帶來的大量背景顏色？將預設修飾符類別替換為.btn-outline-\*刪除任何按鈕上的所有背景圖像和顏色的修飾符類別。

{/* <Canvas of={ButtonStories.ButtonOutline} /> */}

## SIZE

喜歡更大或更小的按鈕嗎？添加.btn-lg 或.btn-sm 以獲得其他尺寸。

{/* <Canvas of={ButtonStories.ButtonSize} /> */}

## 全寬按鈕

結合我們的顯示和間隙實用程序，創建響應式全寬「塊按鈕」堆棧，如 Bootstrap 4 中的按鈕。透過使用實用程式而不是特定於按鈕的類，我們可以更好地控制間距、對齊和回應行為。
